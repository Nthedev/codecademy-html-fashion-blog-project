# Codecademy HTML Fashion Blog Project

This repository is for practicing core HTML skills through building a simple blog post page. The project concept originates from Codecademy's Learn HTML course. Images have been downloaded from Codecademy materials for educational purposes.

## Overview

A fashion blog page titled "Everyday with Isa" featuring an NYC Fashion Week (NYFW) insider's guide. This project demonstrates fundamental HTML concepts including semantic markup, hyperlinks, image embedding, and page organization.

## Technologies Used

- HTML5
- Relative image paths (webp, jpeg formats)

## Skills Demonstrated

- Semantic HTML structure (`h1`–`h4`, `<div>`, etc.)
- Hyperlinks (internal anchors + external links)
- Image embedding with relative paths
- Page layout organization
- Basic content structuring (lists, paragraphs, highlighting)
